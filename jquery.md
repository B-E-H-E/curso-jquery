# jQuery
"Write less, do more"
- Biblioteca de JS
- Converte para JS padrão (ECMAScript)
- Crossbrowser
- Rápido carregamento, captura e transmissão de dados
- Manipula o DOM
- Facilita a consulta (query) de elementos
- Extensível com plugins
- Instalação: https://jquery.com/download/

## Seletores
### Simples
```
    $('h4') //tag
    $('.featured-item') //class
    $('#featured') //id
```
### Compostos
```
    $('h4, h6')
    $('div h4')
```
